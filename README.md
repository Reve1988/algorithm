##### 선택정렬 알고리즘의 특징
비교연산 : O((n-1) + (n-2) + ... + 2 + 1) = O(n^2)

이동연산 : O(3(n-1)) = O(n) 

성능 : O(n^2)

- 상대적으로 느린 알고리즘에 속함
- 자료의 연산 이동횟수가 O(n)이므로 자료의 크기가 큰 정렬에 유리
- 최악의 경우나 평균이나 효율성은 항상 O(n^2)
- 자료의 교환이 계속되므로 안정성이 없음

* 안정성(stability) : 동일한 데이터에 대해 입력한 순서대로 정렬되면 안정성이 있다고 한다
